# 阿軒無盡塔防

[![CI & Deploy Pages](https://github.com/mars-tw/tower-defense-skill/actions/workflows/ci.yml/badge.svg)](https://github.com/mars-tw/tower-defense-skill/actions/workflows/ci.yml)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)
[![Play Online](https://img.shields.io/badge/Play-Pages-brightgreen)](https://mars-tw.github.io/tower-defense-skill/)

原生 HTML/CSS/JavaScript + Canvas 2D 製作的無盡塔防小遊戲。玩家在不同地圖與難度下建塔、施放技能、派出英雄、累積魂晶、抽英雄並挑戰排行榜。

## 主要內容

- Canvas 塔防：路徑地圖、敵人波次、Boss 波、事件波與元素克制。
- 6 種塔：箭塔、砲塔、冰塔、電塔、毒塔、支援塔；升級上限 10。
- 女神核心：生命升級上限 8，Lv.2 後自動聖擊附近敵人。
- 9 種敵人：史萊姆、哥布林、獸人、蝙蝠、冰狼、小惡魔、盾兵、醫者、Boss。
- 元素克制：火克冰、冰克雷、雷克火；克制 1.5 倍，被克 0.66 倍。
- 三難度：普通、嚴酷、無盡；調整敵人 HP、金錢、女神血量與 Boss 間隔。
- 兩地圖：翠綠平原與迂迴峽谷；各自路徑、起始資源倍率與排行榜分榜。
- 首 10 波任務線：建第一座塔、清第一波、英雄上場、第一次升級、第一次技能、清第 3 波、首 Boss、第二英雄。
- 魂晶即時結算：每波清空立刻入袋；死亡結算不再重複補發清波魂晶。
- 英雄系統：10 位英雄、稀有度抽卡、18 抽傳說保底、前 2 位英雄偏好未擁有。
- 英雄指揮：部署小卡顯示頭像/血條/等級，點小卡可進入駐守模式並設定防守半徑。
- 結算摘要：紀錄本局英雄 XP、升級數、成就與地圖分榜名次。
- 快捷鍵：數字建塔、技能快捷、暫停、速度切換、英雄抽卡與進度面板。

## 玩法

1. 選擇難度與地圖，快速開始會使用普通 + 翠綠平原。
2. 建塔守住路徑終點，善用元素克制與毒/支援塔處理特殊敵人。
3. 清波取得金幣與即時魂晶，完成新手任務取得第二抽所需資源。
4. 用魂晶抽英雄，部署後可點英雄小卡設定駐守點。
5. 死亡或結算後查看地圖分榜、成就與英雄成長摘要。

## 關鍵數值

| 類別 | 現況 |
|---|---|
| meta | `td_meta_v1`，`META_VERSION = 7` |
| 起始金幣 | 220，再乘地圖倍率 |
| 塔升級 | 傷害 x1.5、射程 x1.08、費用 x1.52、最高 Lv.10 |
| 女神 | base HP 100、每級 +60、最高 Lv.8、Lv.2 解鎖聖擊 |
| Boss | 普通每 5 波、嚴酷每 4 波、無盡每 3 波 |
| 魂晶 | 普通 1.8、嚴酷 2.4、無盡 2.2 的逐波差分 |
| 英雄抽卡 | 20 魂晶，首抽免費，18 抽傳說保底 |
| 新手任務 | 8 個任務，總獎勵 38 魂晶 |
| 排行榜 | `board[diffId][mapId]`，每榜前 10 |

## 專案結構

| 檔案 | 說明 |
|---|---|
| `index.html` | Canvas、HUD、地圖/難度/排行榜/英雄 UI |
| `src/config.js` | 塔、敵人、技能、女神、地圖、難度、事件波、成就、新手任務 |
| `src/rules.js` | meta v5 遷移、波次生成、逐波魂晶、排行榜與成就規則 |
| `src/heroes.js` | 英雄資料、抽卡、保底、前 2 位未擁有偏好 |
| `src/game.js` | 遊戲迴圈、建塔、技能、英雄、駐守、結算 |
| `src/ui.js` | HUD、任務線、進度面板、英雄小卡與快捷鍵 |
| `references/data-model.md` | 數值與存檔契約 |

## 測試

```bash
npm test
npm run test:e2e
```
